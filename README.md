# Tale Blog

![Tale](https://ooo.0o0.ooo/2017/02/27/58b43450c9182.png)


## 特性

+ 设计简洁，界面美观
+ Markdown 文章发布
+ 自定义文章链接
+ 支持多主题
+ 支持插件扩展
+ 支持 Emoji 表情
+ 支持网易云音乐播放
+ 支持附件和数据库备份
+ 部署简单，不依赖 Tomcat
+ 无需数据库，内嵌 Sqlite

## 如何使用

将代码克隆到本地，执行：
```
git clone https://github.com/tfssweb/tale.git
``` 
然后进入到`docker`目录，依次执行：
```
cp -r ./tfss_volumes/ /opt/

docker-compose up -d
```
即可！
